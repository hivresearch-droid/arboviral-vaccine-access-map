# Arboviral Vaccine Access Map

This repository contains the data and source materials used to derive estimates of arboviral vaccine access presented in the manuscript.

## Contents

- data/country_classification.csv  
  Country-level classification of arboviral transmission and vaccine access.

- sources/source_inventory.csv  
  Curated inventory of data sources.

- docs/methods.md  
  Summary of methods.

## Notes

Data were compiled from WHO, ECDC, PAHO, CDC, and national public-health sources.
